# FinX Client Onboarding Documentation

Internal Docusaurus 3.x documentation site for onboarding clients onto the
FinX platform.

## Local development

```powershell
npm install
npm start
```

The site runs on http://localhost:3000 by default.

## Build

```powershell
npm run build
```

Output is written to `build/`.

## Project structure

- `docs/` — Markdown content, organized by audience (executive, product,
  engineering, partner integration, QA).
- `src/pages/index.js` — Custom landing page.
- `sidebars.js` — Sidebar navigation configuration.
- `docusaurus.config.js` — Site configuration.
- `static/` — Static assets (images, etc.).

## Distribution

Internal documentation. For authorized personnel only.
